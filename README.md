# blog

project to practice & learn React github workflows and aws.


## Start app

```
npm start
```

Navigate to:

```
http://localhost:3000/
```


## tests

```bash
cd my-blog

```

```
npm test

```

## To run a single test suite

```
npx jest -t Navbar
```

or 

```
npm test -- -t="BlogList component"
```

or

```
npm test -- -t="<name of testName>"
```

or

```
npm test -- -p="BlogList.test.tsx"
```
